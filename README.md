## Run Locally

Clone the project

```bash
  git clone https://github.com/GerardoRegalado/todo-app-nextjs
```

Go to the project directory

```bash
  cd todo-app-nextjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
